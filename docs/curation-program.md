# Curation Program

The curation program extends the library in auditable batches:

1. confirm source rows and clean transcript paths;
2. add or revise segment cards;
3. add concept senses;
4. add relation records;
5. add close-reading cards where row-level interpretation needs argument, process, or case detail;
6. update theme dossiers and syntheses;
7. run validation and query smoke checks.

Every batch should preserve exact quote evidence and avoid unanchored summaries.
