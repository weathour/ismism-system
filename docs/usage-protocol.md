# Usage Protocol

1. Start from a row, segment, term, relation, close-reading card, or theme query.
2. Read the cited clean transcript path before making interpretive claims.
3. Keep claims bounded to quoted evidence.
4. Use concept senses for term disambiguation.
5. Use relation records to move between concepts, positions, and evidence.
6. Use close-reading cards for argument, process, and case-level interpretation.
7. Run validators after changing curated assets.
