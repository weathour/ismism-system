# Phase 2：状态跃迁图（首轮）

## 1. 总动力学公式

从已抽出的四条阶段链来看，作者反复使用的是同一个基本动力学：

- `1 = 闭合的一体化`
- `2 = 裂开 / 二分`
- `3 = 中介 / 调和`
- `4 = 再开放 / 跃迁 / 实践化`

这不是抽象猜测，而是分别在四条轴上都能找到对应证据：

- 场域：标准场 -> 分裂场 -> 主体中介场 -> `3+1` 实践场
- 本体论：一元 -> 二分 -> 中心中介 -> 开放四重性
- 认识论：继承 -> 分裂 -> 公共话语调和 -> 开放思辨
- 目的论：有限性 -> 拒绝做中介 -> 失败中调和 -> 呼唤/跳跃

## 3. 轴内通用跃迁模板

### 本体论轴

- `1 → 2`：从“只有一种根本存在”变成“存在被拆成两类/两面”
- `2 → 3`：二分不能自稳，必须引入中心中介项
- `3 → 4`：中心中介仍然过于闭合，于是转向开放四重性

### 认识论轴

- `1 → 2`：不再照单全收既有分类，开始出现知识内部的裂口
- `2 → 3`：裂口要靠公共话语、语用库、合法表达来调和
- `3 → 4`：调和机制再被打开，进入承受荒谬结论的开放思辨

### 目的论轴

- `1 → 2`：从保守有限性转向拒绝做绝对者中介
- `2 → 3`：直接拒绝仍不足，转而通过失败和历险来调和自由与中介
- `3 → 4`：再进一步走到呼唤、跳跃、未来实验

## 5. 组合推理样例

### 样例 A：`场域 2 + 本体论 3`

- 结构含义：
  - 场域 2：背景秩序与场域自身已经裂开
  - 本体论 3：系统又试图用一个中心化项来调和裂口
- 典型后果：
  - 认识论上不太可能继续停留在纯继承式把握
  - 更可能转向主体化、自反化、接口化的认识动作
  - 若异质性他者与自我意识发生短路，就走向辩证法
- 这就是用户问题里“场域阶段2 + 本体论阶段3 会引发怎样的认识论动作”的首轮答案：
  - **它会逼出一种主体化的中介认识论，最典型的走向是自我意识/主体作为接口，进一步可能短路为辩证法。**

## 6. 当前机制结论

- 四阶段不是静态标签，而是一套固定节律：`闭合 → 分裂 → 中介 → 再开启/实践化`
- 在不同轴上，这个节律的具体内容不同，但运动方向高度同构
- Phase 3 可以直接拿这套节律去诊断现实现象是停在闭合、裂口、中介，还是已经逼近实践跃迁
