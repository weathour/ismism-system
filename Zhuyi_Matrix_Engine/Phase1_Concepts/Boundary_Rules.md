# Phase 1：概念边界规则

## 1. 四元轴不能混写

### 场域 != 本体论

- 场域问的是：**存在在什么背景秩序/展开面中出现？**
- 本体论问的是：**在这个展开面里，什么算存在、存在如何分项？**
- 例子：
  - `1-1` 与 `1-2` 共享同一个场域位 `1`
  - 但二者的本体论位分别是 `1` 与 `2`

### 本体论 != 认识论

- 本体论处理存在的构成。
- 认识论处理把握、反思、中介、合法化的方式。
- 例子：
  - `1-1-1` 与 `1-1-3` 共享 `场域=1`、`本体论=1`
  - 但认识论位从 `1` 变为 `3`

### 认识论 != 目的论

- 认识论问“如何知道、如何中介、如何说得通”。
- 目的论问“朝哪里去、为何行动、如何闭合或跃迁”。
- 例子：
  - `3-3-1-1` 与 `3-3-1-4`
  - 前三位相同，只是最后一位从绝对有限性转到呼唤—跳跃

## 2. 判断一段文本属于哪一轴

### 若文本在说……

- `宇宙 / 场 / 符号秩序 / 律令 / 背景秩序 / 历史-人民-行动者`
  - 优先判为 `场域`
- `实体 / 两种时间 / 中心化中介 / 本体单元 / less than nothing`
  - 优先判为 `本体论`
- `前反思 / 反思 / 理性 / 情感 / 公共话语 / 语用库 / 开放姿态`
  - 优先判为 `认识论`
- `伦理实践 / 有限性 / 自由 / 失败 / 呼唤 / 跳跃 / 实验 / 未来`
  - 优先判为 `目的论`

## 3. 同一个数字在不同轴上不是同一个意思

- `场域=4` 不等于 `目的论=4`
- `本体论=2` 不等于 `认识论=2`
- 数字只表示**在该轴内部的阶段位次**
- 所以 `4-2-3-1` 应拆成：
  - 场域四阶段中的第 `4`
  - 本体论四阶段中的第 `2`
  - 认识论四阶段中的第 `3`
  - 目的论四阶段中的第 `1`

## 4. 当前最重要的边界测试

- 如果一句话讨论“世界如何被组织起来”，先看 `场域`
- 如果一句话讨论“什么算真正存在”，先看 `本体论`
- 如果一句话讨论“主体如何把握、表达、调和”，先看 `认识论`
- 如果一句话讨论“系统为何行动、如何收束或跃迁”，先看 `目的论`

## 5. 后续使用规则

- Phase 2 做机制分析时，必须先标注“当前讨论的是哪一轴的跃迁”。
- 任何组合态都必须写成四位编号，不允许只写模糊概念名。
