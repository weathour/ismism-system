# Phase 1：四元轴基础定义

## 1. 结构确认

- 依据 `Zhuyi_Matrix_Engine/Phase0_Corpus/Matrix_Backbone.md`，本书核心目录骨架是非空唯一编号的 `4 × 4 × 4 × 4 = 256`。
- 当前工作假定编号位次就是矩阵位次：
  - 第一位：`场域`
  - 第二位：`本体论`
  - 第三位：`认识论`
  - 第四位：`目的论`
- 这个假定不是外加预设，而是由目录唯一编号统计与一级总论中的编号解释共同支持。

## 2. 四元轴的工作定义

### 场域

- 作者把场域指向“**一切之所在**”，并明确说“把它说成场域已经是一种空间化的隐喻”。
- 因而，场域不是单纯物理空间，而是**让一切存在者得以出现、被容纳、被组织的背景秩序/展开平面**。
- 证据：
  - `split_md/1_实在论/0001_1_实在论_p13.md:141`
  - `split_md/1_实在论/0001_1_实在论_p13.md:145`

### 本体论

- 作者把本体论直接连到“一个场域下的存在者”及其“分类学”。
- 因而，本体论在本书里的功能是：**规定在某个场域里，什么算存在、存在以什么单元/种类/关系出现**。
- 证据：
  - `split_md/1_实在论/0001_1_实在论_p13.md:404`
  - `split_md/1_实在论/0001_1_实在论_p13.md:405`
  - `split_md/1_实在论/0001_1_实在论_p13.md:432`
  - `split_md/1_实在论/0001_1_实在论_p13.md:448`

### 认识论

- 在作者的用法里，认识论不是“抽象的知识论”而已，而是**系统如何把握自身分裂、如何在非反思与反思之间建立中介、如何把某种说法合法化**。
- 在三字头总论里，这一轴被明确关联到：
  - 第一人称体验
  - 先验框架/意识结构/符号学机制
  - 自我意识/主体
  - 公共话语与语用库

- 在作者这里，目的论与伦理实践、意义、行动定向、信仰之跃、未来试验密切相关。

## 3. 当前结论

- `场域` 处理背景秩序/展开面。
- `本体论` 处理存在单元与存在方式。
- `认识论` 处理认知分裂、中介与合法化方式。
- `目的论` 处理行动方向、意义闭合与跃迁姿态。

这四个轴是不同问题，不应互相混写；后续每个条目都必须先判断它在回答哪一个问题。
