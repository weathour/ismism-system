# 模块：诊断树

本模块是 Phase 2 与 Phase 3 的压缩版。详细证据见：

- `Zhuyi_Matrix_Engine/Phase2_Mechanics/State_Transitions.md`
- `Zhuyi_Matrix_Engine/Phase2_Mechanics/Action_Targets.md`
- `Zhuyi_Matrix_Engine/Phase3_Applications/Diagnostic_Anchors.md`

## 1. 一级判定：场域位 `F`

### If 对象……

- 把世界说成天然完整、默认无缝、放之四海而皆准
  - Then `F1`
- 强调隐藏秩序、律令、创世、幕后结构、根本裂口
  - Then `F2`
- 把问题放到主体、自我意识、体验、解释接口上
  - Then `F3`
- 已经进入组织、行动、历史介入，并能指出实践单元
  - Then `F4`

### Hard Rule

- 若自称实践，但无法回答“实践单元是什么”
  - 不能直接判 `F4`

## 2. 二级判定：本体位 `O`

### If 对象……

- 只承认一种根本存在/唯一尺度
  - Then `O1`
- 以两个世界、两种时间、两类存在为底座
  - Then `O2`
- 用某个中心项来调和两边
  - Then `O3`
- 刻意保持开放、多重、空位、未定型
  - Then `O4`

## 3. 三级判定：认识论位 `E`

### If 对象……

- 直接继承既有分类，不反思知识前提
  - Then `E1`
- 把知识拆成两边：感性/理性、有限/无限、情感/认知
  - Then `E2`
- 靠公共话语、语用库、合法表达调和
  - Then `E3`
- 允许悖论、荒谬、反逻辑结论进入推理
  - Then `E4`

## 4. 四级判定：目的位 `T`

### If 对象……

- 以有限性、保守、守成、克制为终点
  - Then `T1`
- 拒绝为绝对者、天命、总体性做中介
  - Then `T2`
- 在失败、历险、建构中推进自由实践
  - Then `T3`
- 以跳跃、试验、未来、高风险为终点
  - Then `T4`

## 5. 机制修正规则

### Rule A：锁死

- If `E` 已出现危机，但对象不重做 `F/O/E`
- And 它诉诸永恒秩序、最终意义、绝对正当性
- Then 标记：`目的论锁死`

### Rule B：短路

- If 对象不断把“异质性他者”直接并到“自我意识/主体”
- Then 标记：`走向辩证法`

### Rule C：伪四字头

- If 对象谈组织/革命/建设
- But 没有实践单元，只有口号或纯理论
- Then 标记：`伪 F4`

## 6. 组合推理规则

### `F2 + O3`

- 默认解释：裂开的场被一个中心项强行调和
- 典型后果：认识论会被迫主体化、自反化
- 默认候选：`E3` 或 `E4`
- 若自我意识与异质性他者短路
  - 优先补记：`辩证法倾向`

### `F4 + O? + E?`

- 一旦进入 `F4`
  - 必须补问：实践单元是谁？
  - 必须补问：它如何进入历史？

### `O4 + E4 + T4`

- 这是高开放、高风险、高实验的结构
- 若缺少实践单元
  - 易滑向空转乌托邦

## 7. 输出前自检

输出前必须检查：

1. 四位编号是否完整？
2. 每一位是否都有对象内部证据？
3. 是否误把不同轴的同一个数字当成同一含义？
4. 若判为 `F4`，是否真的找到实践单元？
