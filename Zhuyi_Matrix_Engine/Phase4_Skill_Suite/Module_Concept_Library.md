## 1. 四元轴

### `F` 场域

- 问题：对象默认的世界背景/展开面是什么？
- 四阶段：
  - `F1` 标准一体场
  - `F2` 裂开的场
  - `F3` 主体中介场
  - `F4` 实践性的 `3+1` 场

### `O` 本体论

- 问题：在这个场域中，什么算存在？存在如何分项？
- 四阶段：
  - `O1` 一元实体化
  - `O2` 二分本体论
  - `O3` 中心中介化
  - `O4` 开放四重性

### `E` 认识论

- 问题：对象如何把握自己？如何处理认知裂口？
- 四阶段：
  - `E1` 继承式/不反思认识
  - `E2` 分裂认识
  - `E3` 公共话语调和
  - `E4` 开放思辨

### `T` 目的论

- 问题：对象朝哪里去？如何闭合、行动或跃迁？
- 四阶段：
  - `T1` 绝对有限性
  - `T2` 拒绝充当绝对者中介
  - `T3` 失败中调和的自由实践
  - `T4` 呼唤—跳跃 / 未来实验

## 2. 基本动力学

默认阶段节律：

- `1 = 闭合`
- `2 = 分裂`
- `3 = 中介`
- `4 = 再开启 / 跃迁 / 实践化`

## 3. 关键术语

- `符号秩序`：背景规则/律令维度
- `异质性他者`：与主体/自我意识形成张力的异物
- `公共话语`：认识论三阶段的主要调和器
- `实践单元`：四字头成立的必要条件
- `锁死`：用目的论把已暴露问题的结构重新封闭
- `短路`：异质性他者与自我意识直接连通，系统滑向辩证法

## 4. 最小判定问题集

面对对象，先问四句：

1. 它默认世界是什么样的？
2. 它承认几种存在方式？
3. 它靠什么证明自己知道？
4. 它最后想把人和世界带向哪里？

答完这四句，再给编号。
