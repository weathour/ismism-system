# 0325_4-3-1-3_列阵的统一_p8187
左边是只管生产，自给自足，生产维持自身，自给自足，自己维持自身，自我维持生存。右边就是组织生产，组织生产活动。右边是要这个生产活动当中，生产关系所产生的一些异化。有人买单，负责要负责。这两者之间肯定会发生关联，就是说发生一个冲突。就是说它实际上是具有外部性的。就是这里面那个冲突，就是你知道什么叫统一战线。你这里你就听懂了，你要让他负责这些人，那是什么人？他是地主，还有开明士绅，对不对？开明士绅就是地主了。开明士绅这个人要让他们负责，目的论是要让这些负责，让这些清空，其实是你，你让他们负责。就是你间接的，他们直接的要负责，要出钱出粮，要认购，要合作，要负责，要合作。他们要合作。在这里就有个问题，也就是说如果他就不合作，他如果不和你联合，他和外面那些联合怎么办？那这两个就是冲突的。我所以你明白就说，你强迫要求他们为他们的那种前现在的生产关系，或者说异化的生产关系所造成那些所谓的经济的外部性，要让他买单。
