他对约伯记的解读你可以自己去了解一下，我待会儿会讲。舍斯托夫认为这个世界的场域也是……他是一个比较温和的，但也是看上去最激进的，就是要把……就是哲学不是解释性的活动，哲学不是……就是哲学等于生存、等于绝望下的抗争，只有绝望下的抗争。但这种抗争是一种完全手脚被捆住、完全没有任何能力抗争的，只有智性抗争，只有智性和意志的迸发，意志的一个迸发，只剩下一个精神状态了，绝望下的抗争。这个绝望就是手脚都锁起来，就是陀思妥耶夫斯基里面写的那个关在石头屋子里面，然后用头撞墙这种状态，身体都已经完全绝望了，就是面对真正就是输了、已经是输了，就按照必然性，抗争什么？那抗争就是必然性，对于必然性的抗争，是真正的必然性，就是真正的本体论秩序的抗争，就是绝望下的抗争，这个抗争是对于本体论秩序。不是本体性的秩序，而是本体论的那些秩序的一个抗争。比如说生死这种大的秩序，生死、时间的流逝、时间的线性流逝，然后就是主体性的内在性，就这些大的，还有比如说痛苦的内在的他者性，就是不可否弃性，就是这些本体论上大的秩序，客体的实在性、异质性，本体论上这些秩序。
