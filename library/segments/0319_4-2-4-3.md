# 0319_4-2-4-3_集体创新_p8030
这种创新还有另外一个意思，就是communalization这个东西、这个过程，它以新的形式发生出来，以全新的形式发生出来的。这个全新形式是没有人给他下指导棋的，无人下指导棋，没有人教你怎么去弄。而是你自己从劳动中、从过程中、从这个活动过程中，从这个当中去寻得它的那个理，寻找它那个理。这个理是发生的、发生出来，理是发生出来的。所以他在认识论上的这个，就是游离。这个理不是说先有理后再去做，而是在做的过程中实事求是，在行动中、在活动中发生，它自己发生出来、自己结构出来，不需要知识分子下指导棋，不需要知识分子给你下指导棋的。这一点在朗西埃对于阿尔多塞的批评上面是，就是对于他老师、和他老师的决裂当中，也不叫决裂，他老师那条路走不通了。就是文本，没有说知识分子的文本就在那个时代，哲学家的文本就高、就了不。就就可以指导他就是这个原文本，而其他所有的下面的文本都是残次品、劣等的，不是的。文本符号化的活动、符号过程、符号学机制，就整个 class consciousness，它的折射反应不是说一定要到知识分子的反思当中才是最高级的形态。
