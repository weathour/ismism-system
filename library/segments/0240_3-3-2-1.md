# 0240_3-3-2-1_安放了的实存主义_Positioned_Existentialism_克尔凯廓尔的三个阶段与两种辩证法_p6167
就是他找到自己位置的、定位了的实存主义，或者我们叫……positioned这个词翻译怎么翻比较好……安置、安放了的实存主义，有所安放的、安放的、被安放的，甚至是可以看成是被安放的实存主义。因为克尔凯郭尔的这个辩证法，这个辩证法的梳理我主要借鉴的是齐泽克，当然还有瓦尔特·舒尔茨。我们都算晚期谢林主义者，就从观念论上讲，算黑格尔主义者加晚期谢林主义者，所以我们用的那套行话都是一样的。克尔凯廓尔的哲学就是三阶段论，非常典型的审美、伦理、最后是宗教三阶段论。舒尔茨用观念论的术语去把握的话，因为克尔凯郭尔是一个后观念论，所以他的分析单元是实存。实存就是人的生存，而且侧重的是内在体验、真实内在体验、本真的内在体验。在审美维度的话，他被叫做一种无方向的在自身中。审美维度……我们就要一个个来讲了。审美维度一定会超越到伦理维度，伦理维度一定会超越到宗教维度。
