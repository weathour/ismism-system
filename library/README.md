# ISMISM Library Assets

`library/` contains curated, validator-backed assets derived from the corpus.

- `manifests/`: corpus manifest, segment manifest, chunk manifest, and anomaly notes.
- `segments/`: row-level segment cards.
- `concepts/`: concept senses, core concept notes, and ambiguity notes.
- `positions/`: matrix position cards.
- `relations/`: relation graph records and relation guides.
- `close-reading/`: argument, process, and case cards.
- `themes/`: theme dossiers with row manifests and evidence banks.
- `syntheses/`: synthesis documents.
- `protocols/`: reading and diagnosis protocols.
- `templates/`: templates for new curated assets.
- `audits/`: durable validation and evidence audit records.
