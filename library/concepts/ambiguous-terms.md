## High-risk ambiguity found

### 主体

- 已拆出 4 个 draft 义项：矩阵主体侧、先验/反思主体性、主体间性机制、无意识/驱力主体。
- 高风险：`主体`、`主体性`、`主体化`、`去主体化` 在 row 262/276 等处发生机制性交叠；后续必须单独处理 `主体化` 与 `去主体化`。
- 禁止：把所有“主体”都解释成自主意识或个人行动者。

### 客体

- 已拆出 4 个 draft 义项：矩阵客体/实体侧、现象学意向对象、小客体/客体小a、异化/物神化客体。
- 高风险：`客体` 在 row188 的 noema、row46 的客体小a、row344 的 object/fetish 三处不是同一层。
- 禁止：把客体小a或社会物神化对象当作普通外部对象。

### 实践

- 已拆出 5 个 draft 义项：目的论行动姿态、第四主部历史-政治实践场、人与物实践关系、实践单元/可执行载体、避免异化的运动式主动活动。
- 高风险：`实践` 同时是一级场域名、目的论姿态、物关系结构和具体执行单元；必须保留 row/toc 定位。
- 禁止：把 `实践单元` 当成 `实践` 的总定义。

### 人民

- 已拆出 5 个 draft 义项：人民性、人民中介位、人民理论化、天然理论家、人民可数单元。
- 高风险：`人民`同一条内含“场域中介位—理论化机制—普罗大众生活—可数单元”四类语义。
  - 行动风险：将“人民作为中介位”与“人民作为天然理论家”混为一体；
  - 结构风险：把 `历史`、`理论`、`人民`三元关系中的可替代位点误当作固定顺序；
  - 指称风险：把“人民可数性”当作人口统计学概念。
- 禁止：将 `人民` 简化为单一民族/群众意志或单指“受压迫者”道德身份；
  - 禁止将“人民理论家”与已独立义项 `理论家` 在语法上互相替换。

### 历史

- 已拆出 5 个 draft 义项：时间性历史范式/物质史、场域链路中的中介候选、历史的哲学化/辩证法化、史学研究实践、历史的自我妥协与现实化。
- 高风险：`历史`与“人民—历史—理论家”链条紧耦合，且与实践场/理论家姿态/史料共同体容易重叠；后续需与人民/理论批次对齐。
- 禁止：把历史还原为事件叙述、把“历史哲学化”当成纯历史事实复述、或把历史妥协误读成政治保守。

### 机制词（主体化、客体化、去主体化）

- 已将 `主体化`、`客体化`、`去主体化` 在 B5 统一拆分，当前均为 `draft` 机制项，且证据来自 2-2-4/2-2-2-2/4。
- 高风险：
  - 三者在文本中常连续出现，容易被误读为同一术语或同一机制的不同阶段；
  - `主体化` 与 `去主体化` 易与“无意识主体性”互相挤占，导致与 `主体:s04` 重叠；
  - `客体化` 与 `历史` 的“本体论化/现象对象化”语义重叠，易混入 `客体` 的客体位项；
  - 在 `理论` 段位，三者与“生产性”“客体客体化”等命题并行，需在 relation 明确关系方向（produce/transform/condition）。
- 禁止：
  - 不要把三者当作可交换同义词；
  - 不要把“去主体化”理解成单纯“主体消亡”；
  - 不要把“主体化”当成固定“人格增强”或“修辞强化”。

### 理论

- 已拆出 5 个 draft 义项：中介位、符号系统性/历史性、理论生产性、理论-现实回路、理论现实化-通俗化。
- 高风险：
  - `理论`在 row 276 与 `理论家`/`历史`/`人民`形成三元耦合，容易与“理论家化”混淆；
  - `符号系统性`含“历史性”与“排他性”双向表述，常与“实践性有效性”交叉；
  - `理论生产性`与 `客体化`、`主体化` 机制词高度重叠，后续在关系层要拆机制项。
- 禁止：
  - 不要把理论直接理解为“真理本体”或“最终行动者”；
  - 不要把通俗化理解为低层传播技巧；
  - 不要把 `理论` 与 `理论家` 在语法上互相替换。
