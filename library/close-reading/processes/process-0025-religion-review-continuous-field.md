# Close-reading card: process 0025 religion review continuous field

## Card Function

本卡把 row 25 `1-2` 纳入 Religion Problem theme package。它读取的是 ISMISM 内部的宗教问题功能，不是外部宗教学通论。[q1] [q2] [q3] [q4]

## Process Skeleton

1. 定位：本行在主题分类 `religious-realism-cosmos` 中承担 `宗教实在论把一字头世界场读成由最高秩序主宰的 cosmos。` 的功能。[q1]
2. 机制：引文显示该行如何把 sacred / faith / idol / spirit / fetishism / salvation / ideology 的某一环节转成可追踪结构。[q2]
3. 边界：后续引文防止把局部宗教功能扩张为全书或外部宗教史判断。[q3]
4. 余项：最后一条引文保留语气、例证或转折，供 synthesis 只按证据引用。[q4]

## Preconditions and Failure Modes

- 前提：先回到 `religion-evidence-bank.jsonl` 和 declared clean text，再使用本卡。
- 失败模式：把关键词命中自动当成核心宗教论，或把 close-reading pilot-draft 当作 canonical 摘要。
