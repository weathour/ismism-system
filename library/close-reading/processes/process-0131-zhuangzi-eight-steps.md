# Close-reading card: process 0131 zhuangzi eight steps

## Card Function

本卡记录 row 131 中的过程型知识：它不是术语义项，也不是人物条目，而是一个分阶段修行/主体改造协议。

## Process Skeleton

1. **外天下**：解除庸俗实在论对“天下/整体世界”的直接把握，把世界总体现成性从当然前提中移开。[q2]
2. **外物**：脱掉把对象固定为客体的方式，松动主体对物的客体化关系。[q3]
3. **外生**：继续解除对生命、自我保存和经验连续性的执着；本 pilot 只把它作为八步序列中的第三步记录，详细释义需下一批补证。[q1]
4. **朝彻**：转入更透明的反思状态；本 pilot 只把它作为八步序列中的第四步记录，详细释义需下一批补证。[q1]
5. **见独**：作为八步序列中的第五步保留，具体操作含义需在后续 close-reading 扩展中补充 quote-level 证据。[q1]
6. **无古今**：作为八步序列中的第六步保留，具体去时间化释义需在后续 close-reading 扩展中补充 quote-level 证据。[q1]
7. **不死不生**：作为八步序列中的第七步保留，具体生死二元释义需在后续 close-reading 扩展中补充 quote-level 证据。[q1]
8. **撄宁**：作为八步序列中的第八步保留；本卡不把其详细黑格尔式/实践式展开当作已完成解释。[q1]

## Preconditions and Failure Modes

- 前提：读者必须把八步理解为可操作的坐忘/去结构化程序，而非单纯文献训诂。[q4]
- 失败模式：若把“外”理解为逃避世界，或把步骤理解为抽象玄谈，就会丢失文本强调的实操性。

## Downstream Use

适合后续课程 traversal、实践训练案例和“过程卡”格式迭代；不适合直接替代《庄子》原典解释。
