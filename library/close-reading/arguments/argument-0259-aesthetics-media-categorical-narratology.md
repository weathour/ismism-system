# Close-reading card: argument 0259 aesthetics media categorical narratology

## Card Function

本卡把 row 259 的材料吸收到 Aesthetics / Art / Media / Image / Narrative 主题层。它服务于未来电影分析矩阵，但当前只记录语料内部的结构功能。

## Close-Reading Skeleton

1. **主题入口**：本行把电影、影像、艺术、美学、文本或叙事材料作为哲学结构的显影面，而不是外部影评材料。[q1]
2. **媒介机制**：相关词项把观看、符号、图像、文本或故事组织成可追踪的中介结构。[q2]
3. **主体位置**：主体在这里通常作为观众、叙事角色、解释者或被媒介化的欲望位置出现。[q3]
4. **分析边界**：本行可供电影分析借用，但必须回到 row/quote，而不能把例子扩展成泛化电影学公式。[q4]

## Failure Modes

- 把电影例子当作外部影片事实，而不是 ISMISM 的结构类比。
- 只抓“电影/艺术/诗/叙事”关键词，却不区分美学、媒介、影像、文本、意识形态与实践层级。
- 直接套用精神分析、资本主义或女权主题，而没有说明本行的影像/叙事功能。

## Evidence Quotes

- 它不是真正的叙事层。它否则文学不成其为文学。文学不是一个个的这个事件人称化，然后动起来就是文学的。文学要让你感觉到他在讲述一个世界，那么能够让你感觉到讲述、编织起一个世界的是要靠这个叙事层。这个叙事层在这里面有叙事者，就是作者，作者在里面施加的一些结构、一些关联，还有读者和现实意识形态之间
- 就作者已经死掉了，叙事者只是一个符号学拟制。但是无论如何，叙事学使得一种分类变得可能，就是人称化的叙事和非人称化的一个叙事变得可能了。这个人称化叙事它是在……我们就要具体来看他的叙事学了，分类性的叙事学。那么具体而讲，这个分类学在叙事学本身的架构里面是分层、分层性的一个叙事学。就分层也是一
- 符号系统。这个姿态是罗兰巴特通过写专栏对于社会现象进行嬉笑怒骂……也就在这个意义上讲，罗兰巴特所追求的零度写作，所谓的zero level，零度文学、零度写作（“writing degree zero”），他最终使得一切都变成写作、使得一切都变成文本了，everything is……一切都
- 一个零度、零阶维度，就是一切都是一个共时性的、差异性的能指系统、符号系统在进行这种无限的排列组合的游戏。而且其他的所有的政治、经济、其他的文化、艺术这些东西，生活具体的生活、人生体验、感知思维、反思哲学、精神分析，所有这些东西都同样处在一个符号系统的零度层。所以在本体论上是使得叙事广义化了
