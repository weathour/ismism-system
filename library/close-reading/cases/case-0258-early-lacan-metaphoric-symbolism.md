# Close-reading card: case 0258 early lacan metaphoric symbolism

## Card Function

本卡记录 row 258 的 case-level 吸收：早期拉康不是作为完整精神分析体系被重述，而是作为 3-4-1-2 中“隐喻的象征主义”的样例。

## Case Positioning

- 对象：早期拉康，尤其镜像阶段、镜像认同与象征秩序之间的过渡。
- 关键判断：文本认为早期拉康的理论态势仍是“隐喻性的象征主义”，实在界尚未成为强问题域。[q1][q4]

## ISMISM Reconstruction

1. 镜像不是经验物件“镜子”的简单功能，而是主体在外部关系中遭遇的模型关系。[q2]
2. 镜像关系被本段推进为一种特殊能指关系，因此不只是心理发展图像。[q3]
3. 若镜像认同没有进一步整合进父性的象征秩序，就会两极震荡；本卡不把攻击性/自恋性分支当作已完全展开的 case 结论。[q5]
4. 因此，本段把早期拉康理解为由想象界/镜像问题向象征秩序纵深化的过渡型案例。[q1][q4]

## Standard-View Contrast

本卡不试图裁定拉康研究史；它只保存本段的内部判断：镜像模型是通向象征主义的隐喻装置，而不是一个孤立心理发展描述。
