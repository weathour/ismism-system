# Close-reading card: case 0173 john stuart mill

## Card Function

本卡把 row 173 的人物/学派定位整理为 case card：核心不是给穆勒做百科词条，而是记录 ISMISM 如何把他放入 2-4-2-1 的自然主义规范论。

## Figure / School Positioning

- 人物：John Stuart Mill / 约翰·穆勒。
- 定位：实证主义邻近的自然主义规范论，但文本把它诊断为“自然主义的唯心主义”或“伪装成自然主义的观念论”。[q1]
- 矩阵位置：2-4-2-1，强调反形而上学姿态背后仍有形而上学背景秩序。

## ISMISM Reconstruction

1. 穆勒看似经验主义，因为他强调归纳法和经验观察；本段将其重写为“只相信归纳法的康德”。[q3]
2. 但文本明确把穆勒从经验主义改判为观念论者。[q2]
3. 功利主义中的 happiness / pleasure 不是私人主观快乐，而是被共同体中介、比较、规训过的跨主体规范。[q4]
4. 因此，本卡把所谓自然主义规范论理解为观念论式共同体规范结构，而非纯经验事实。[q1][q2][q4]

## Standard-View Contrast

通常叙述会把穆勒放在经验主义、自由主义或功利主义谱系中；本卡只记录本段的反向读法：经验主义外观遮蔽了规范共同体的观念论结构。

## Evidence Quotes

- 我把它叫做自然主义的唯心主义、naturalistic idealism，甚至我可以写成是伪装成自然主义的观念论pseudo-naturalistic idealism。
- 穆勒不是经验主义者，穆勒就是观念论者。
- 穆勒是一个相当于只相信归纳法的康德。
- 所以它的 pleasure 不是 subjective 的。It is not subjective。
