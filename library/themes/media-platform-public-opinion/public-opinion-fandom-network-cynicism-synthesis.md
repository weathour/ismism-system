# Public Opinion / Fandom / Network Cynicism Synthesis

- status: G016 focused synthesis

The public-affect side of the layer links public opinion windows, public forgetting, public discourse space, fan/idol relations, traffic idols, internet celebrity face, network cynicism, and judgment. Its minimal chain is: event visibility → opinion window → public affect → fan/idol or cynical identification → judgment or forgetting.

This layer should not be used as a moral complaint about online life. It is a row-grounded diagnostic grammar. `粉丝`, `爱豆`, `流量小生`, `网红脸`, `舆论窗口`, `公众遗忘`, `公共话语空间`, `舆论审判`, `网络犬儒`, and `义愤填膺观看` are usable only when exact quotes support the step being claimed.

For later film analysis, this gives a way to read crowds, screens, celebrity, scandal, trial, spectacle, live comment, and public emotion as structures of mediated subjectivation rather than merely as plot devices.
