# Aesthetics / Art / Media / Image / Narrative Taxonomy

This taxonomy controls the film-analysis precursor theme. A row is included only when the evidence shows an aesthetics, art, media, image, viewing, symbol, text, fiction, poetry, narrative, cinema-example, or mass-culture function.

## Core classes

- `consciousness-stream-cinematic-experience` — world/experience as film-like flow.
- `aestheticism-gaze-fantasy` — aesthetic posture, gaze, fantasy, and visuality.
- `decadent-aesthetics-poetry` — decadent aesthetics, poetry, symbol, and style.
- `poetry-metaphysics` / `poetic-ontology` — poetry as metaphysical or ontological form.
- `phenomenological-aestheticism` — phenomenology supporting aesthetic experience.
- `symbolic-form-myth-art` — symbolic forms, myth, art, and expression.
- `picture-logic-image` — image/picture logic and visual registration.
- `fictional-existentialism` / `poetic-eternalism-novel-narrative` — fiction, novel, text, dream, and narrative temporality.
- `categorical-narratology` / `local-totality-narrative-media` — narratology and media-totality rows.
- `hermeneutics-text` / `phenomenological-hermeneutics-text` / `language-games-symbol` — text, rhetoric, interpretation, and rule-bound meaning.

## Bridge classes

Bridge rows are valid only when they show how movie examples, viewing, image, narrative, mass culture, or media connect to psychoanalysis, capitalism, feminism, religion, time/death, AI, or practice.

## Context classes

Context rows preserve useful examples but do not drive theme claims by themselves.
