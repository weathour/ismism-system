# Film Analysis Precursor Synthesis

- status: bridge synthesis for future matrix design
- do not use as a finished film-analysis matrix

## Recommended future bridge order

1. Start with Aesthetics-Media for the film's image, viewing, narrative, text, genre, and media form.
2. Add Psychoanalysis-Subjectivity for desire, fantasy, symptom, gaze, unconscious, big Other, and discourse.
3. Add Capitalism for commodity, industry, class, consumption, labor, finance, and spectacle.
4. Add Feminism for gendered body, erotic economy, love ideology, family, reproduction, and patriarchal fantasy.
5. Add Time-Death for memory, finitude, death, repetition, immortality, and technical life.
6. Add Religion or AI when sacred order, salvation, idolization, technical life, robot subjectivity, or algorithmic world-building is explicit.

## Why this layer matters

The previous recommendation named the strongest existing themes. This absorption adds the missing hinge: rows where ISMISM already treats cinema, image, art, poetry, fiction, text, narrative, media, and viewing as theoretical operations. Without this hinge, film analysis would jump too quickly from a movie to psychoanalysis or capitalism and miss the image/narrative form itself.
