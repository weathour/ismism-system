# 现象学 taxonomy

- status: controlled vocabulary for Field 3 theme package
- source boundary: corpus/registry/toc.csv rows 187-275 and cited clean markdown
- rule: each manifest row has exactly one theme_class in this taxonomy

## Classes
- phenomenology-overview-review: 3-1 入口与复习课，界定现象学事业、对手和内部谱系。
- transcendental-intentionality-reduction: 先验现象学、意向性、先验还原、noesis/noema 与主体/客体二分的重写。
- eidetic-factual-phenomenology: 实事/本质现象学，把本质还原、价值、情感和历史性作为方法问题。
- lifeworld-intersubjectivity: 生活世界、主体间性、共同世界与主体性替代方案。
- worldly-ontological-phenomenology: 现世、性化、身体、知觉、行动和存在论现象学。
- phenomenology-bridge: Field 3 其他主题中显式触及现象学方法或人物的桥接行。
- excluded-keyword-only: 关键词出现但不支撑现象学主题功能的控制行。
