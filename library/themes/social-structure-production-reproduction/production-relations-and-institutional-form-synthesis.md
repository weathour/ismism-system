# 生产关系与制度形式专题综合

生产关系类证据（row 321, row 322, row 323, row 324, row 327, row 331, row 342）提示：制度形式不是生产之外的壳，而是生产活动、生产者、产品、对象与关系的稳定方式。核心风险是把生产力抽象为效率，把制度抽象为善恶，从而遗漏关系本身如何被组织、如何异化、如何被合法化。

使用方式：先查 `--class production-relations-institutional-form`，再用 state/governance 与 reproduction 类交叉验证。制度形态只有在它实际固定生产/再生产关系时才进入本主题。
