# 社会再生产与经济生活专题综合

社会再生产类证据（row 32, row 61, row 64, row 65, row 90, row 114, row 134, row 183, row 250, row 257, row 265, row 275, row 280, row 295）显示，家庭、照护、妇女/儿童、生活必需品和日常经济组织是结构持续的基础层。再生产不是“私人生活”对“公共制度”的补充，而是生产关系能够被下一轮主体继续承接的机制。

使用方式：当分析社会黑暗、门派/家族、朝廷/宗族、婚姻/生计时，必须问：谁承担再生产成本？谁被训练成可支配主体？生活资料、照护义务和名分如何被制度化？
