# 阶级—空间—意识形态—金融桥接综合

阶级、空间、教育/意识形态、金融和消费并不是外围话题。它们分别把结构位置显影为通道、地域、资格、名望、信用、账本、货币、商品和生活方式。相关证据包括 row 60, row 63, row 76, row 84, row 86, row 174, row 184, row 232, row 245, row 287, row 289, row 298, row 305, row 311, row 312, row 313, row 314, row 315, row 97, row 99。

使用方式：结构分析要避免“只讲朝廷黑暗”或“只讲资本贪婪”。需要追问：空间如何分配机会，教育如何排序主体，金融/账本如何远程控制，消费和生活水准如何把结构转化成日常欲望。
