# 国家—官僚—合法性专题综合

国家/官僚/合法性类证据（row 9, row 12, row 19, row 21, row 47, row 50, row 59, row 82, row 83, row 87, row 92, row 108, row 300, row 304, row 309, row 310）提示：规则、程序、合法性、审判、命令和行政组织常常把社会关系重新包装成“秩序”。本主题不把国家或制度当作单纯压迫机器，也不当作中性容器，而看其如何中介生产/再生产关系。

使用方式：分析任何朝廷、门规、官僚集团、地方秩序时，先区分：程序合法性、实质正义、再生产利益、阶级/资本位置和组织自保逻辑。
