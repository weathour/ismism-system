# Rural-Urban Difference and Public Space Synthesis

- status: G028 focused synthesis

城乡差异不是“城市好/乡村坏”的价值判断，而是生产方式、土地关系、公共可见性和主体组织方式的差异化空间。公共空间也不是单纯开放场所，而是主体能否出现、如何被观看、如何被治理、如何被排除的社会载体。

## Main oppositions

| Pair | Diagnostic question |
|---|---|
| 城市 / 农村 | 哪种生产—生活组织塑造主体位置？ |
| 公共空间 / 私人居住 | 主体是公开显现还是退回安顿/不安？ |
| 土地 / 房子 | 空间是生产基础还是居住财产想象？ |
| 迁移 / 安顿 | 主体能否稳定占据一个位置？ |
| 空间化 / 实践 | 关系是否被固定成位置、等级和路线？ |

## Boundary

不要把城乡差异直接等同于现实城乡政策，不要把公共空间直接等同于城市规划，不要把住房标签直接等同于房价或租房建议。
