# Urban / Housing / Migration / Space Synthesis

- status: G028 theme package synthesis
- scope: 城市 / 房子 / 居住 / 农村 / 乡村 / 城乡差异 / 土地 / 迁移 / 漂泊 / 公共空间 / 空间阶层化
- evidence: `urban-housing-migration-space-row-manifest.jsonl` + `urban-housing-migration-space-evidence-bank.jsonl`

## Core thesis

ISMISM 的城市—住房—迁徙—空间主题并不是房地产市场、规划学或迁徙政策资料库。它把“空间”读成社会矛盾的承载形式：城市让主体进入公共可见性，房子与居住把安顿问题物化，土地和农村显示生产关系的地理基础，迁移/漂泊暴露主体位置的流动化，而公共空间与空间阶层化说明社会秩序如何把人、物、行动路线和价值分配到不同位置。

## Diagnostic chain

1. **城市/公共空间**：城市不是中性容器，而是公共话语、消费、治理、资本和主体显现共同发生的场。
2. **房子/居住**：房子在文本中首先是居住物与安顿位置；只有在精确引文支持下，才可进一步接到住房焦虑、买房或财产化想象。
3. **农村/土地/城乡**：农村、乡村和土地不是怀旧背景，而是生产、农业、资本化和现代化重排的空间证据。
4. **迁移/漂泊**：迁移和漂泊把主体从固定位置中抽离，显示城市生活、身份位置和共同体归属的失败或转移。
5. **空间化/阶层化**：空间化把关系转换成位置；空间阶层化则让等级、资本和公共性差异在位置安排中沉淀。

## Boundary

`住房`、`租房`、`房价`、`户籍`在 G026 clean-text 搜索中没有精确命中；它们只能作为查询端或后续电影/社会分析端的下游标签，不能作为独立证据。通用的形而上学 `空间` 也不能自动升级为城市/住房分析。
