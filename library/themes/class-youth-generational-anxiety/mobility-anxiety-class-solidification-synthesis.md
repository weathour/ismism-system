# Mobility Anxiety and Class Solidification Synthesis

- status: G022 closure sub-synthesis
- scope: 阶层 / 上升通道 / 阶层固化 / 中产焦虑 / 教育阶层 / 地位 / 流动

## Diagnostic chain

The mobility layer is organized by a contradiction: social discourse promises movement, but the corpus repeatedly ties movement to education, cultural capital, income, labor price, and inherited symbolic position. `上升` is therefore not treated as an abstract direction; it is a social promise whose carriers can close back into hierarchy.

```text
education/culture/labor carrier → promise of upward movement → ranking and competition → blocked or narrowed channel → anxiety and self-blame
```

## Key distinctions

- `阶层` names the lived position; `阶级` names the historical/political-economic relation.
- `中产` and `小布尔乔亚` mark fragile cultural/educational positions rather than stable prosperity.
- `上升通道` is a promise; `阶层固化` is the practical experience of the promise failing or narrowing.
- `地位` is diagnostic only when it marks social recognition, not when it is merely logical or ontological status.

## Boundary

This synthesis does not assert an external theory of contemporary mobility. It records how this corpus makes mobility anxiety diagnosable: by linking education, labor, culture, money, and symbolic recognition into a row-traceable mechanism.
