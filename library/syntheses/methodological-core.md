# synthesis Synthesis — Methodological Core

- status: draft synthesis
- scope: how to use the matrix as an auditable interpretation method; every substantive claim below is source-tagged.

## Method sequence

- First disambiguate terms, because high-risk terms such as `主体`, `客体`, `理论`, `现实`, `星丛`, and `Event` have multiple row-specific senses.[row 46, term:主体:s01, position 1-3]
- Then locate the matrix position, because a term’s function changes when it appears in a field, ontology, epistemology or teleology coordinate.[row 46, term:客体:s01, position 1-3]
- Then inspect relation assets, because transitions, tensions, mediations and misrecognitions should use controlled predicates rather than loose association.[row 285, term:中介:s02, position 4-1-2-1, relation rel:w5b5:001]
- Then synthesize only with source tags, because synthesis claims must remain traceable to segment, term, position and relation evidence.[row 285, term:严肃:s01, position 4-1-2-1]

## Operational terms already in concept

- `观察` supplies situated material rather than detached neutrality.[row 285, term:观察:s02, position 4-1-2-1]
- `分析` is concurrent with observation and also includes inner reflection after conversation.[row 285, term:分析:s02, position 4-1-2-1]
- `回溯` moves from detail to structural source and then back to concrete reality.[row 285, term:回溯:s02, position 4-1-2-1]
- `现实理论化` converts reality into a conceptual/structural form that can be replayed or predicted.[row 285, term:现实理论化:s02, position 4-1-2-1]
- `理论现实化` converts theory into a describable or practical force in reality.[row 285, term:理论现实化:s01, position 4-1-2-1]
- `可行性` keeps interpretation tied to possible action rather than self-contained commentary.[row 285, term:可行性:s01, position 4-1-2-1]

## Guardrails

- The method must not treat external generated material as canonical because concept and relation validation depend on corpus/clean-markdown evidence and segment/position paths.[row 285, term:研究:s01, position 4-1-2-1]
- The method must not collapse all senses of a term into one definition because audit concept-drift audit found the sense boundaries useful for high-risk terms.[row 229, term:名称:s01, position 3-2-4-2]
- The method must not turn relation assets into universal necessity because audit relation audit preserves applicability boundaries and forbidden interpretations.[row 263, term:矛盾张力:s01, position 3-4-2-2, relation rel:w5b4:008]
- The method must keep concept/relation records draft until an explicit promotion audit exists.[row 285, term:严肃:s02, position 4-1-2-1]

## Minimal query path

- For a term question, read `term-senses.jsonl`, then `core-terms.md`, then the cited clean row.[row 229, term:名字:s01, position 3-2-4-2]
- For a position question, read the position card, then the associated segment segment card, then the concept terms used by that card.[row 263, term:点位:s01, position 3-4-2-2]
- For a movement question, read relation relation assets, then verify both source and target positions, then check source senses.[row 263, term:数据交换中介:s02, position 3-4-2-2, relation rel:w5b2:011]
