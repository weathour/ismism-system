# synthesis Synthesis — Part 3 Idealism / 观念论

- status: draft synthesis
- scope: field 3 positions and concept/relation links; every substantive claim below is source-tagged.

## Core movement

- 观念论场域集中处理符号系统、主体性、否定辩证法和事件哲学如何把“存在”转成可争夺的结构。[row 263, term:Being:s01, position 3-4-2-2]
- 在静止事件主义中，`Being` 与 `Event` 构成在场/不在场和服务器正常运行/本体论版本更新的场域对立。[row 263, term:Event:s02, position 3-4-2-2]
- `site` 与 `position` 标记事件发生学点位和主体性坚持的位置，而不是普通空间地点。[row 263, term:site:s01, position 3-4-2-2]
- 阿多诺的 `星丛` 把黑格尔式闭合辩证法消解不掉的符号化剩余保留下来。[row 229, term:星丛:s01, position 3-2-4-2]

## Mechanism and terms

- `回溯性逻辑` 在 row 263 中说明一种更新痕迹被遮蔽、变化被回收成“本来如此”的逻辑。[row 263, term:回溯性逻辑:s02, position 3-4-2-2]
- `本体论更新` 把 Event 读成版本更新，并通过主体化/site 的外部更新中介传送。[row 263, term:本体论更新:s02, position 3-4-2-2]
- `无调性` 在 row 229 中不是单纯音乐风格，而是离散符号系统和否定辩证法形而上学架构的说明方式。[row 229, term:无调性:s02, position 3-2-4-2]
- `不可还原性` 标记主体内在客体性和否定辩证法面对的失败界限。[row 229, term:不可还原性:s02, position 3-2-4-2]

## Boundary notes

- 观念论综合不能把主体性简单肯定化，因为 row 263 同时记录了主体性作为 site/position 机制位和作为空间化/康德式主体性的批判对象。[row 263, term:主体性:s02, position 3-4-2-2]
- 观念论综合不能把数学话语当作无中介真理，因为 row 263 把数学话语描述为可被本体论化、哲学化和占位的材料。[row 263, term:数学话语:s01, position 3-4-2-2]
- relation 的 `misrecognizes-as`, `overcodes`, `objectifies` 关系适合追踪观念论内部误认和超编码，但这些关系仍为 draft。[row 263, term:数学化:s01, position 3-4-2-2, relation rel:w5b3:012]
