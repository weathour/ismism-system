## 位置定义
3-3-3 是生存论内部的认识论三位。row 244 以尼采为核心，将同一生存论组织为永恒轮回、权力意志、生命和价值的结构：时间序列被降格，过去/未来被理解为同一者自我同一运动中的符号化差异。

该位置的中介不再是克尔凯郭尔式抉择等级制，而是“爱命运”。生命作为免于意义束缚的主体性审视价值，价值则是权力意志自我中介中的剩余或产物。爱命运意味着主体承担自身作为中介性的命运，并把有限角色理解为世界自身世界化的过程。

## 代表主义/学派
- 尼采式同一生存论、永恒轮回、权力意志
- 爱命运、价值重估、生命哲学、超人线索
- 舍斯托夫等后续尼采主义/抗争生存论候选

## 核心机制
1. **永恒轮回**：不是经验时间轮转，而是同一者自我同一和无时间结构。
2. **生命/价值区分**：生命是权力意志的基础性力量，价值是其中介剩余。
3. **爱命运**：主体把自身作为中介性来承担、定义和创造命运。
4. **去人称化**：超人不是普通人称角色，而是免于意义框架束缚的主体性过程。

## 与相邻位置的关系
- 相对于 3-3-2，3-3-3 不再停留于有限主体与绝对者的间接关系，而以生命/权力意志接管中介。
- 向 3-3-4 移动时，命运和中介性会转入叙事文本、虚构世界和符号身份。
- 向 3-4 移动时，价值剩余与符号化机制会被符号学结构进一步明确化。
- 向 4 移动时，爱命运若要转化，需要进入历史实践而非只保留生存姿态。
