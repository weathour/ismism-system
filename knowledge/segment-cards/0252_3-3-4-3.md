# 0252_3-3-4-3_幸福的幸存主义_p6490
他也接受不了苏维埃制造出来的那些痛苦。它是一个调和状态。纳粹这边就是伊万、宗教大法官，苏维埃就是K。然后加缪是在这里调和。调和就是那个西西弗斯。这个调和者的形象就是西西弗斯。调和，他的反抗是调和性的。加缪那边的反抗，荒谬本身都是调和型的，它是调和性的。只不过这个东西，所以在认识论上就会对应四。我们来看加缪的一个场域论的基本……加缪的小说你自己去看，什么《鼠疫》、什么《正义者》、《西西弗斯》。在场域论上是三字头。三字头左边是理性。理性就是那个主体化的理念、等级制、理念秩序在这里。这个左边这东西是比较蛇皮的，是加缪比较蠢的一点，他的幻想主体化的有一个，就是它的荒谬不来自于主体性本身，它的荒谬来自于主体和世界间的张力。这是它哲学比较废的地方。所以它不是哲学家……vs世界的敌意。就像萨特，萨特那边他人是地狱，然后自我是他人、自我原初的是他人。这是萨特荣居三杠三杠一的一个原因。而加缪他在哲学上有点肤浅，他的世界的敌意，就世界超出于这个理性，存在超出于理。
