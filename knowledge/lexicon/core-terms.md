## 主体

### term:主体:s01 — 矩阵中的主体侧维度

- axis: 复合
- status: draft
- confidence: high
- definition: 在 ISMISM 四轴框架中，“主体”首先可指认识论与目的论所代表的主体性维度：把握、反思、中介、行动准则和伦理闭合的侧面。它与场域论/本体论对应的客体或实体维度相区分，但这种区分只是矩阵读法，不是实体—主体的绝对二分。
- source rows: row 46 `1-3`
- forbidden mix: 不要把该义项当成经验个人、心理自我或政治行动者；它是矩阵轴读法中的“主体侧”，并且不得被固定为实体/主体二分。

### term:主体:s02 — 先验/反思主体性

- axis: 认识论
- status: draft
- confidence: medium
- definition: 在现象学与后康德语境中，“主体”可指认识活动的反思条件、常项或先验自我维度，例如我思、纯粹统觉、先验还原中的构造位置。这个义项强调能否作为认识与世界构成的条件，而不是日常人格。
- source rows: row 176 `2-4-2-4`; row 188 `3-1`; row 193 `3-1-1-3`
- forbidden mix: 不要把先验/反思主体性混同为主体间性机制、政治代理人或无意识驱力主体；这里的核心是认识论条件/构造位置。

### term:主体:s03 — 主体间性/人称化机制

- axis: 认识论
- status: draft
- confidence: high
- definition: 在生活世界现象学和哈贝马斯路径中，“主体”被转写为主体间性：认识起点不再是孤立的先验主体，而是人称化、视域可置换与视域交织所形成的共同中介机制。
- source rows: row 200 `3-1-3`; row 201 `3-1-3-1`
- forbidden mix: 不要把主体间性直接还原为单个自我、先验自我或群体政治主体；这里的义项落在可交换视域和人称化机制。

### term:主体:s04 — 无意识/驱力主体与去主体化

- axis: 复合
- status: draft
- confidence: high
- definition: 在拉康—齐泽克及后结构主义语境中，“主体”可指被符号系统、阉割、欲望/驱力结构生产出来的无意识或驱力主体；其关键运动是去主体化，甚至转成历史的客体。
- source rows: row 262 `3-4-2-1`; row 276 `4`
- forbidden mix: 不要把该义项解释成自主理性主体或日常行动者；它是符号化/阉割/去主体化过程中的位置，且常与客体化交叠。

## 客体

### term:客体:s01 — 矩阵中的客体/实体侧维度

- axis: 复合
- status: draft
- confidence: high
- definition: 在 ISMISM 四轴框架中，“客体”可指场域论与本体论这一侧的实体/背景/存在构成维度；它是矩阵读法中的客体侧，不等同于某个具体物。
- source rows: row 46 `1-3`
- forbidden mix: 不要把该义项混同为意向对象、客体小a、异化对象或被物神化的社会客体；这里是四轴结构中的客体/实体侧。

### term:客体:s02 — 意向对象/noema 的客体极

- axis: 认识论
- status: draft
- confidence: high
- definition: 在现象学语境中，“客体”是意向结构里的对象极或 noema；普通主客二分被批评为不够严格，因为主体、客体与背景性域常构成三分关系，而胡塞尔试图把它压缩为意向行为/意向对象的二分。
- source rows: row 188 `3-1`; row 217 `3-2-2`
- forbidden mix: 不要把现象学意向对象直接当作社会异化物、欲望剩余或矩阵客体侧；该义项需与 noesis/noema、主客关系和背景域一起读。

### term:客体:s03 — 小客体/客体小a

- axis: 复合
- status: draft
- confidence: medium
- definition: 在欲望辩证法语境中，“客体”可专指小客体/客体小a：主体遭遇他者或剩余时的欲望对象位，围绕否认、对抗、投射、厌恶等策略展开。
- source rows: row 46 `1-3`
- forbidden mix: 不要把客体小a等同于普通外部对象、意向对象或结构客体侧；它是欲望/剩余/他者遭遇中的对象位。

### term:客体:s04 — 异化/物神化中的对象化客体

- axis: 复合
- status: draft
- confidence: high
- definition: 在实践与社会关系语境中，“客体”可指被对象化、客观化或物神化的关系/人/享乐/历史位置。它常与去主体化和客体化相连，也可被乌托邦式实践要求解除其固定对象地位。
- source rows: row 276 `4`; row 344 `4-4-1`
- forbidden mix: 不要把异化/物神化客体误读为现象学 noema 或简单实物；这里强调关系被固定为对象、主体被历史客体化的机制。
