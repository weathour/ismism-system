# w10:arg:0076:contemporary-naturalism

## Card Function

本卡把 row 76 的强烈批判性讲法拆成可复用的论证骨架。它不改写 W3/W5，也不把 W2 摘要替换为新摘要。

## Critique Target

- 对象：当代自然主义及其在英美分析哲学中的常识化、日常语言化和科学共同体依附形式。
- 主要问题：文本认为该立场在本体论上把自然/科学共同体当作默认背景，在认识论上要求一致性与可通约性，从而让哲学变成服务、澄清、编辑既有体系的工作。

## Reconstructed Argument Steps

1. 先把“当代自然主义”定位为 1-4 平庸主义内部的子型，而不是中性哲学流派名。[q1]
2. 再指出它的本体论姿态不是积极论证自然，而是接受日常语言和常识结构中既有的本体论预设。[q2]
3. 接着把这种姿态扩展到学术共同体机制：它倾向于澄清、维系、服务既有科学/学术体系，而不是批判体系自身。[q3]
4. 最后把问题收束为反思权力的弃守：不是“科学解释一切”的胜利，而是哲学把本体论反思外包给权威。[q4]

## Rhetoric Boundary

文本中“帮佣”“平庸”等词属于 polemical register。W10 复用时应保留其诊断方向，但不要把修辞强度误当作已完成的社会学统计。

## Evidence Quotes

- 当代自然主义是英美分析哲学的普遍的本体论立场、普遍的认识论立场
- 他们认同在日常语言结构里面的本体论的预设
- 他们的学术共同体机制不批判，学术体系他们是不批判的
- 他们把自己的这个反思本体论的这个权力扔掉、放掉了。
