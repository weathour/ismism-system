# w10:arg:0065:aesthetics-media-aestheticism-gaze

## Card Function

本卡把 row 65 的材料吸收到 Aesthetics / Art / Media / Image / Narrative 主题层。它服务于未来电影分析矩阵，但当前只记录语料内部的结构功能。

## Close-Reading Skeleton

1. **主题入口**：本行把电影、影像、艺术、美学、文本或叙事材料作为哲学结构的显影面，而不是外部影评材料。[q1]
2. **媒介机制**：相关词项把观看、符号、图像、文本或故事组织成可追踪的中介结构。[q2]
3. **主体位置**：主体在这里通常作为观众、叙事角色、解释者或被媒介化的欲望位置出现。[q3]
4. **分析边界**：本行可供电影分析借用，但必须回到 row/quote，而不能把例子扩展成泛化电影学公式。[q4]

## Failure Modes

- 把电影例子当作外部影片事实，而不是 ISMISM 的结构类比。
- 只抓“电影/艺术/诗/叙事”关键词，却不区分美学、媒介、影像、文本、意识形态与实践层级。
- 直接套用精神分析、资本主义或女权主题，而没有说明本行的影像/叙事功能。

## Evidence Quotes

- 它出现了这个诡异之物，它具有一种符号学的能力、自组织的能力。它这种视差性重复意味着诡异之物具有符号学上的自组织能力。它就组建了一个审美的符号体系、审美化的符号学体系。在这个审美符号体系里，平凡之物就会被定义为丑，然后这个诡异之物就会被定义成beauty，定义成美。
- §1. 克尔凯廓尔的审美-伦理-三阶段论。克尔凯廓尔主义的一部分，他的审美是他三阶段论之一：审美-伦理-宗教。实际上克尔凯廓尔三阶段不是这样的，而是：审美vs伦理，这是第一阶段。人处在一个享乐（享受审美）和伦理性的对立，也，享乐和禁欲的这个二元对立处，在这种对立之中的是审美阶段，或者叫审美
- 呢。额头基本上就是鼻头，额头对应的这个鼻头，这两个是同构的。额头要明亮饱满但又不能太长，因为鼻头它突出的这个……鼻头它后面就是通道了。所以在日式审美里面，日本还不是把这边剃掉了，这不太可能。后面这个长辫子为什么漂亮，应该是直接就连上这个通道了，差不多是一个对应物的一种关系。然后继续要讲，从
- 我出于我个人的一个审美偏好，想给你们讲一个概念：高跟鞋就是血肉崇拜。高跟鞋它有它的审美必然性，还有那个stockings它也是有它的审美必然性的。而且它们都是一个很有意思的互补关系，是倒数关系。我们直接先来讲这个高跟鞋了。高跟鞋最经典的两种颜色是红色和黑色，红色代表着嘴唇，黑色代表着头发。
