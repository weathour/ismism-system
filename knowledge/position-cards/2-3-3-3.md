## 位置定义
逻辑原子主义是单子主义下的目的论三位。它以罗素分析实在论为代表，把世界理解为可由最优语言和逻辑结构衡量的事实原子；这种分析姿态表面反形而上学，实则预设逻辑结构已渗透现实。

## 代表主义/学派
- 逻辑原子主义
- 罗素分析实在论
- knowledge by acquaintance / description 线索

## 核心机制
1. 用逻辑语言衡量事物中的确定性。
2. 把现实拆分为可分析的事实或逻辑原子。
3. 以分析结构延续单子主义的封闭单位模型。

## 与相邻位置的关系
- 相对于 2-3-3-2，它减少体系宏观分类，强化语言和逻辑单位。
- 相对于 2-3-3-4，它仍相信分析结构能稳定把握现实。
- 它把单子主义转入现代分析实在论的形而上学预设。
